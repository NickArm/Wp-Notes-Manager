A powerful collaborative notes and task management system for WordPress. Perfect for teams, project management, and personal productivity.

== Description ==

Notes Manager transforms your WordPress site into a collaborative workspace where users can create, assign, and manage notes and tasks efficiently. Whether you're running a team blog, managing projects, or need a simple task management system, this plugin provides all the tools you need.

= Key Features =

* **Collaborative Notes System** - Create notes that can be assigned to specific users
* **Flexible Note Types** - Add notes to dashboard, posts, or pages
* **Stage Management** - Customizable workflow stages (To Do, In Progress, Done, etc.)
* **User Assignment** - Assign notes to team members with proper permissions
* **Priority Levels** - Set priority levels (Low, Medium, High, Urgent)
* **Advanced Filtering** - Filter notes by author, assignment, stage, or type
* **Multiple Layout Options** - Choose between list, 2-column, or 3-column views
* **Audit Trail** - Complete logging of all note activities
* **Responsive Design** - Works perfectly on desktop, tablet, and mobile
* **Admin Dashboard Integration** - Seamless integration with WordPress admin
* **Security First** - Proper nonce verification and permission checks

= Perfect For =

* **Team Blogs** - Assign tasks and track progress
* **Project Management** - Organize projects with stages and assignments
* **Content Teams** - Manage editorial workflows and assignments
* **Small Businesses** - Simple task management without external tools
* **Personal Productivity** - Organize your own tasks and notes

= User Roles & Permissions =

* **Administrators** - Full access to all features, stage management, audit logs
* **Editors** - Can create, edit, and manage notes
* **Authors** - Can create and edit their own notes
* **Contributors** - Can create notes (pending review)

= Stage Management =

Create custom workflow stages like:
* To Do
* In Progress
* To Be Discussed
* Done
* Canceled

Each stage can have custom colors and descriptions.

= Layout Options =

Choose how to view your notes:
* **List View** - Traditional single column for detailed reading
* **2 Columns** - Balanced view for better overview
* **3 Columns** - Quick overview for power users

= Security & Privacy =

* All AJAX requests use proper nonce verification
* User permissions are strictly enforced
* Only note owners can edit/delete their notes
* Complete audit trail for accountability
* No external API calls or data sharing

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-notes-manager` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Notes menu in your WordPress admin to start creating notes
4. Configure stages in Notes > Stages (admin only)
5. View audit logs in Notes > Audit Logs (admin only)

== Frequently Asked Questions ==

= Does this plugin work with any WordPress theme? =

Yes! WP Notes Manager is designed to work with any WordPress theme. The admin interface uses WordPress's native styling, and frontend elements are designed to be theme-agnostic.

= Can I use this for project management? =

Absolutely! The stage management system, user assignments, and priority levels make it perfect for project management. You can create custom stages for your workflow and assign tasks to team members.

= Is there a limit to the number of notes? =

No, there's no limit to the number of notes you can create. The plugin uses efficient database queries and pagination to handle large numbers of notes.

= Can I export my notes? =

Currently, notes are stored in the WordPress database. Future versions may include export functionality.

= Does this work with multisite? =

Yes, WP Notes Manager works with WordPress multisite installations. Each site will have its own separate notes system.

= Is my data secure? =

Yes! All data is stored in your WordPress database. No external services are used, and all security best practices are followed.
